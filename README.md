# Atlas Company

Landing page institucional da Atlas Company — sites profissionais entregues em até 5 dias para dentistas, advogados, clínicas, corretores de imóveis e outros negócios sem presença digital.

Site estático de página única (HTML + [Tailwind CSS](https://tailwindcss.com/) via CDN), sem build ou dependências. Basta abrir `index.html` no navegador para visualizar localmente.

## Deploy

Publicado na [Vercel](https://vercel.com) direto a partir deste repositório — todo push na branch `main` gera um novo deploy automaticamente.
